cc_quit_request

Synopsis

    #include "cltsrvr.h"
    
    pid_t cc_quit_request( nid_t cc_node );

Description

cc_quit_request() attempts to contact either the CmdCtrl or PBreg processes to request that a quit proxy be sent at the appropriate time.

Returns

On success, the PID of the quit proxy is returned. On failure, if nl_response is set low enough to prevent an abort, -1 is returned.

See Also

kbdclt_quit().
