Col_reset_proxy

Synopsis

    
    #include "collect.h"
    int Col_reset_proxy(unsigned char id);

Description

Col_reset_proxy() instructs the TM collection program to cease sending to the indicated proxy and detaches the proxy.

id is the magic number agreed-upon between the process and collection for identifying which proxy is designated.

Returns

Returns 0 on success. On error, returns -1 unless nl_response indicates a fatal action.

See Also

Data Collection functions, Col_set_proxy().
