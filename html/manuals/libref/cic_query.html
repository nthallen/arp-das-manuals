cic_query

Synopsis

    #include "nortlib.h"
    int cic_query(char *version);

Description

cic_query queries the command server for version information. If successful, the version of the server is written into the buffer pointed to by the version argument.

Returns

Returns zero on success.

Command Server and Client functions.
