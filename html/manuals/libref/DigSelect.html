DigSelect

Synopsis

    #include "nortlib.h"
    
    unsigned short DigSelect( unsigned short cmd, unsigned short val );

Description

Sends the specified SELECT command to DCCC. This function should work with SET commands as well.

Returns

DigSelect() returns the value argument passed to it.

See Also

send_dascmd(), Client/Server functions.
